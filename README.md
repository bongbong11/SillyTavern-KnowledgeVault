# 📚 지식창고 (Knowledge Vault)

SillyTavern용 확장 프로그램 - 캐릭터 인지 단계별 지식 관리 시스템

## 🎯 핵심 기능

- **5단계 인지 시스템**: Lv.0(봉인) ~ Lv.4(진실)
- **자동 단계 생성**: AI가 영어 원문을 5단계로 자동 요약
- **세계관/페르소나 분리**: 두 가지 타입의 지식 관리
- **간단한 UI**: 마법봉 메뉴에서 접근

## 📦 설치

1. 이 폴더를 `SillyTavern/public/scripts/extensions/knowledge-vault`에 복사
2. SillyTavern 재시작
3. Extensions 탭에서 "Knowledge Vault" 활성화

## 🚀 사용법

1. **설정**: Extensions → Knowledge Vault 설정에서 Connection Manager 프로필 선택
2. **지식 추가**: 마법봉 메뉴(🪄) → 📚 지식창고 → 세계관/페르소나 탭에서 추가
3. **자동 생성**: 제목, 키워드, 전체 내용 입력 → "5단계 자동 생성" 클릭

## 💡 참고

- 지식 내용은 영어로 작성해야 합니다
- AI 자동 생성은 Connection Manager 프로필을 사용합니다
- 생성된 단계는 수동으로 수정 가능합니다
